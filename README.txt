A modern Gutenberg block plugin for embedding YouTube and Vimeo videos with custom player controls and aspect ratio options.

== Description ==

**VidPress** is a **Gutenberg-only** video block plugin that provides a customizable video player supporting YouTube and Vimeo.

It allows you to easily embed videos in your WordPress posts or pages with essential playback features such as autoplay, loop, and aspect ratio. No shortcodes, no Classic Editor — just clean, modern block integration.

**Key Features:**

- Fully Gutenberg-based plugin
- Supports YouTube and Vimeo
- Autoplay and loop toggle options
- Aspect ratio control (16:9, 4:3, 1:1, etc.)
- Lightweight and responsive player
- Custom frontend UI for videos
- Built with performance in mind
- Translation-ready

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/vidpress` directory, or install it through the WordPress plugin repository.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. In the block editor (Gutenberg), search for **"VidPress – Video"**.
4. Add the block to your post or page and customize the settings.

== Frequently Asked Questions ==

= Is VidPress compatible with the Classic Editor? =
No. VidPress is built exclusively for the Gutenberg block editor and does not support the Classic Editor or shortcodes.

= What platforms are supported? =
You can embed videos from **YouTube** and **Vimeo**.

= Can I customize the player layout or controls? =
Yes. VidPress provides options like autoplay, loop, and aspect ratio for layout customization.

= Is the block responsive? =
Yes. The video block adapts automatically to all screen sizes and devices.

= Can I style the block using my theme? =
Definitely! You can add custom CSS or use class selectors to override the default styling.
